Local Weather Project

updates | Mar 20, 2016

Project was creating a weather app that gets your current location via browser. I used the recommend weather app API OpenWeatherMap. Relatively straight forward project. Few days ago I did the project with the random quote API, so that was very helpful here. Parsing the JSON took a short learning curve but otherwise not very troublesome.

Really starting to get a very comfortable feel with CSS. CSS was probably the more intimidating of the HTML frontend stuff. It always seems like a fight to get something the way you want it. Lately that doesn’t seem to be the case.

Well here it is and my code will always be available on my github : Live Local Weather App

CSS Frustrations /rant-warning

Things like clearfix on everything, negative margins and in simple things like fighting a solution to center stuff.

Just seemed like everything was a ‘hacky’ solution. Things are getting a lot more clear for me and I am sure that in part to be able to exclusively use CSS3 and ditch a lot of the old hacky stuff due to browser compatibility but also getting a better understanding of the basics have been great.

Next few projects will have more focus on making things look much cleaner and more attractive.
